El apoyo MIoT a la Formación


Salvamento Marítimo lanza FormaciON, un ambicioso proyecto de innovación con el que se busca incorporar el Internet de las Cosas para mejorar la comunicación entre alumnos e Instructores en las prácticas de Formación.

EL OBJETIVO

El fin de la Aplicación es la gestión con Internet de las Cosas (Managemente with IoT - MIoT) de las comunicaciones o elementos de las Sesiones de Formación donde los participantes llevan dispositivos o las áreas disponen de equipamiento para mejorar el proceso de aprendizaje, transmitir información de forma clara y recolectar datos importantes.

LA APP

Basada en software y librerías de código abierto, completamente autónoma y segura, sobre una aplicación web responsiva y adaptable a cualquier dispositivo. Optimizada para navegadores como Chrome, Safari, Edge o Firefox.

LAS PRÁCTICAS

Los instructores enviarán mensajes u órdenes a los dispositivos, y quedarán registrados en un proceso completamente integrado.

LOS CONTROLES

Desde las pantallas de control se podrá interactuar con el equipamiento de formación
